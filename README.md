# Design Patterns - Weekly Solutions

Weekly solutions for design pattern exercises, implemented in Java.

## Structure

```
design-patterns-solutions/
└── Week1/
    ├── Singleton/
    └── Factory/
```

Each new week's work will be added as a new `WeekN/` folder at the top level, keeping every week's solutions self-contained and easy to find.

## How to run any example

Every pattern folder is self-contained (no external dependencies). From inside that folder:

```bash
javac *.java
java <NameOfDemoClass>
```

See each week's own README for the exact class name to run.
